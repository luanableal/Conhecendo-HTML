Conhecendo o HTML

O que é um HTML?

Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos

O que são tags?

As tags são usadas para informar ao navegador a estrutura do site. Ou seja: quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página.

Principais tags HTML:

  • Parágrafo: p
  • Títulos: h1 até h6
  • Quebra de linha: br
  • Criar listas não ordenadas: ul
  • Criar linhas(ítens) para sua lista: li
